# 🐍 Aplicação Rápida em Python - Estácio

Este repositório foi criado para armazenar e organizar meus estudos e exercícios práticos da disciplina de Python no curso de **Análise e Desenvolvimento de Sistemas**.

## 🚀 O que estou praticando aqui:

Nesta etapa, foco na manipulação de arquivos externos e tratamento de dados textuais:

- **Manipulação de Arquivos:** Uso de `with open()` para leitura e escrita segura de dados.
- **Tratamento de Strings:** Aplicação de métodos como `.strip()`, `.split()` e `.join()` para limpeza e organização de informações.
- **Depuração (Debug):** Uso da função `repr()` para visualizar caracteres invisíveis e garantir a integridade dos dados brutos.
- **Versionamento:** Uso de Git e GitHub para controle de versões do código.

## 🛠️ Tecnologias Utilizadas:
- Python 3.x
- VS Code (Ambiente de desenvolvimento)
- Git/GitHub (Controle de versão)
