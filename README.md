# 🐍 Desenvolvimento Rápido de Aplicações em Python

Este repositório contém as atividades práticas desenvolvidas durante o **3º Semestre** do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O foco principal é a criação de aplicações eficientes, seguras e bem estruturadas utilizando a linguagem Python.

---

## 📂 Organização por Aulas

Navegue pelos módulos de aprendizado clicando nos links abaixo:

*   **[📁 Aula 03 - Manipulação de Arquivos e IO](./Aula_3)**  
    Estudo aprofundado sobre abertura de arquivos (`w`, `w+`, `a`), navegação de cursor com `seek(0)` e tratamento de erros de sistema como `FileNotFoundError` e `PermissionError`.

*   **[📁 Aula 04 - Tratamento de Exceções e Lógica de Login](./Aula_4)**  
    Implementação de lógica robusta com blocos `try/except/finally`, validação de dados de usuário e uso da palavra-chave `raise` para regras de negócio personalizadas.

---

## 🛠️ Tecnologias e Conceitos Dominados

*   **Linguagem:** Python 3.x
*   **Tratamento de Erros:** Gestão de exceções para evitar o fechamento abrupto de programas.
*   **Segurança de Dados:** Validação de entradas e verificação de credenciais.
*   **Versionamento:** Controle de versão profissional utilizando Git e GitHub.
